Server:
- Server wird bei "zu wenig joins" langsamer.
- Bei Anmeldung eines Spielers wird score an alle Spectators übertragen
- Spieler können sich auch wieder abmelden.
- Server in Testmodus setzen: Langsamer und ohne Zeitlimit der Clients.

Java Client:
- Meldet sich beim stoppen ab
- Beendet sich bei Server-Timeout

Spectator:
- Beendet sich bei Server-Timeout (oder versucht sich dananch neu zu registrieren)

Python Client:

- Doku für Python Bot Programmierer.

- Skript um Server und Spring-Spectator gemeinsam neu zu starten.
